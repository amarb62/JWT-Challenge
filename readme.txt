*Following steps are needed to be taken to run the API*

1. Install all the dependencies. Run the following command on terminal to do this:
    npm install express bcrypt jsonwebtoken body-parser

2. Open two different terminals and execute following commands
    a) node client.js
    b) node auth.js

3. Open your browser and go to the following url:
    http://localhost:5000/

4. Click on Sign in with google.

5. Type the following credentials and then click on Sign in button
    Username: amarb
    Password: 1234

6. Then a page should display 'Hi amarb' and that means the api are working perfectly.
